Atribut Boolean
  • 1. Checked
    Makna: Menandai elemen <input> tipe checkbox atau radio sebagai tercentang atau terpilih secara default.
  • Contoh:
    <input type="checkbox" checked> Checkbox ini tercentang
    <input type="radio" checked> Radio button ini terpilih
  • 2. Disabled
    Makna: Menonaktifkan elemen sehingga tidak dapat digunakan atau diinteraksi oleh pengguna.
  • Contoh:
    <button disabled>Tombol ini dinonaktifkan</button>
    <input type="text" disabled placeholder="Input ini dinonaktifkan">
  • 3. Readonly
    Makna: Membuat elemen input hanya bisa dibaca tanpa bisa diubah oleh pengguna.
  • Contoh:
    <input type="text" readonly value="Hanya bisa dibaca">
    <textarea readonly>Teks ini tidak bisa diedit</textarea>
  • 4. Required
    Makna: Menjadikan elemen input wajib diisi sebelum formulir dikirim.
  • Contoh:
    <input type="text" required placeholder="Harus diisi">
  • 5. Autofocus
    Makna: Otomatis memfokuskan elemen input saat halaman dimuat.
  • Contoh:
    <input type="text" autofocus placeholder="Input ini mendapat fokus">
  • 6. Multiple
    Makna: Mengizinkan pengguna memilih lebih dari satu file atau opsi.
  • Contoh:
    <input type="file" multiple>
    <select multiple>
      <option>Opsi 1</option>
      <option>Opsi 2</option>
    </select>
  • 7. Hidden
    Makna: Menyembunyikan elemen dari tampilan pengguna tanpa menghapusnya dari DOM.
  • Contoh:
    <div hidden>Ini elemen tersembunyi</div>
  • 8. Autoplay
    Makna: Memutar media (audio/video) secara otomatis saat halaman dimuat.
  • Contoh:
    <video src="video.mp4" autoplay></video>
  • 9. Loop
    Makna: Memutar ulang media secara otomatis setelah selesai.
  • Contoh:
    <audio src="audio.mp3" loop></audio>
  • 10. Controls
    Makna: Menampilkan kontrol media seperti play, pause, dan volume pada elemen audio/video.
  • Contoh:
    <audio src="audio.mp3" controls></audio>
    <video src="video.mp4" controls></video>
  • 11. Selected
    Makna: Menandai opsi pada elemen select sebagai pilihan default.
  • Contoh:
    <select>
      <option>Opsi 1</option>
      <option selected>Opsi 2 (terpilih)</option>
    </select>
  • 12. Novalidate
    Makna: Menonaktifkan validasi form bawaan browser saat dikirim.
  • Contoh:
    <form novalidate>
      <input type="email" placeholder="Masukkan email">
      <button type="submit">Submit</button>
    </form>
  • 13. Open
    Makna: Menampilkan detail terbuka secara default pada elemen <details>.
  • Contoh:
    <details open>
      <summary>Judul Detail</summary>
      Isi dari elemen detail.
    </details>
  • 14. Reversed
    Makna: Membalik urutan daftar terurut (`
      `).
    1. Contoh:
      <ol reversed>
        <li>Item 3</li>
        <li>Item 2</li>
        <li>Item 1</li>
      </ol>
    2. 15. Spellcheck
      Makna: Mengaktifkan pengecekan ejaan.
    3. Contoh:
      <input type="text" spellcheck="true" placeholder="Cek ejaan aktif">
    4. 16. Download
      Makna: Menyediakan link untuk mengunduh file.
    5. Contoh:
      <a href="file.pdf" download>Download File PDF</a>
    6. 17. Async
      Makna: Menjalankan skrip secara asynchronous.
    7. Contoh:
      <script src="script.js" async></script>
    8. 18. Defer
      Makna: Menunda eksekusi skrip hingga halaman selesai dimuat.
    9. Contoh:
      <script src="script.js" defer></script>
