Bab 1: Pendahuluan

    1.Apa itu HTML dan JS?
  • HTML (HyperText Markup Language) adalah bahasa yang digunakan untuk membuat struktur dari halaman web.
  • JavaScript adalah bahasa pemrograman yang memungkinkan kita menambahkan interaktivitas pada halaman web, seperti animasi, validasi formulir, dan reaksi pengguna terhadap elemen.
    • 2.Fungsi dan Kegunaan HTML & JavaScript
  • HTML berfungsi untuk menampilkan teks, gambar, video, dan konten lainnya dalam struktur yang terorganisir pada halaman web.
  • JavaScript digunakan untuk membuat halaman lebih interaktif dan responsif, memungkinkan pengguna untuk berinteraksi secara langsung dengan elemen di halaman.

Bab 4: Interaksi HTML dengan JavaScript

1. Mengakses dan Mengubah Elemen HTML JavaScript memungkinkan kita untuk mengakses elemen HTML dan mengubahnya. Salah satu metode yang paling umum adalah document.getElementById. JavaScript memungkinkan kita untuk berinteraksi dengan elemen HTML di halaman web. Salah satu cara untuk mengakses elemen HTML tertentu adalah dengan menggunakan metode document.getElementById(). Metode ini akan mencari elemen HTML berdasarkan nilai atribut id dan mengembalikannya sehingga bisa dimodifikasi oleh JavaScript.

Contoh mengubah konten HTML

Penjelasan:

1. Tag P dengan ID "paragraf":

  • Tag P ini memiliki atribut id="paragraf", yang berfungsi sebagai identitas unik. Dengan ID ini, JavaScript dapat mencari elemen ini secara spesifik di dalam halaman.

2.Fungsi JavaScript ubahTeks():

  • Setelah elemen ditemukan, kita mengubah kontennya menggunakan .innerHTML = "Teks telah diubah!". Properti innerHTML memungkinkan kita untuk mengganti konten HTML yang ada di dalam elemen tersebut.

3. Tombol button onclick="ubahTeks()">Ubah Teks button:

  • Tombol ini menggunakan atribut onclick="ubahTeks()", yang berarti saat tombol ini diklik, fungsi ubahTeks() akan dipanggil.
  • Ketika fungsi dipanggil, JavaScript akan mencari elemen p dengan id="paragraf" dan mengubah teks di dalamnya menjadi "Teks telah diubah!".
